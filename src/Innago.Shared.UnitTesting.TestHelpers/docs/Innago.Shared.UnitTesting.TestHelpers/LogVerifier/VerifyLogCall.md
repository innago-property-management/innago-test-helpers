# LogVerifier.VerifyLogCall&lt;TCategory&gt; method

Verifies that a Log call has been made.

```csharp
public static void VerifyLogCall<TCategory>(this ILogger<TCategory> logger, 
    LogLevel expectedLogLevel, string expectedMessagePattern, Exception? exception = null, 
    StringMatchOption matchOption = StringMatchOption.Equals, 
    ITestOutputHelper? testOutputHelper = null)
```

| parameter | description |
| --- | --- |
| TCategory | The logger category. |
| logger | The ILogger. NOTE: The logger MUST BE Mock-created. |
| expectedLogLevel | The expected LogLevel. |
| expectedMessagePattern | The expected message pattern. |
| exception | The expected Exception. This is relevant for Object[]). |
| matchOption | The [`StringMatchOption`](../LogVerifier.StringMatchOption.md). |
| testOutputHelper | The ITestOutputHelper. If supplied, the message patterns seen will be logged. |

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | *logger* instance is not created by Moq. |

## Examples

```csharp
var logger = Mock.Of<ILogger<MyClass>>();
var instance = new MyClass(logger);
instance.DoSomething();

logger.VerifyLogCall(
    LogLevel.Information,
    "MyClass: ",
    matchOption: LogVerifier.StringMatchOption.StartsWith);
```

```csharp
logger.VerifyLogCall(
    LogLevel.Error,
    exception.Message,
    exception,
    matchOption: LogVerifier.StringMatchOption.EndsWith,
    testOutputHelper: outputHelper);
```
